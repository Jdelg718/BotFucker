# Code of Conduct

This project welcomes people who want to make email automation safer, more transparent, and less annoying.

## Expected Behavior

- Be respectful and direct.
- Assume good intent, but be willing to improve ideas.
- Keep discussions focused on behavior, evidence, and user safety.
- Respect privacy. Do not post private email content or contact data.

## Unacceptable Behavior

- Harassment or personal attacks
- Posting secrets, private emails, or private contact lists
- Encouraging abusive automation
- Contributions designed to target people rather than unwanted message patterns

## Reporting

If something in the project feels unsafe or hostile, open an issue with context. If the matter involves private information, avoid posting sensitive details publicly.
